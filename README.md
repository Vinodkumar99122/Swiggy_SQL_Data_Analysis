🍽️ **Swiggy SQL Data Analysis Project**

## Project Overview

This project analyzes Swiggy restaurant data using SQL.
It covers database creation, data manipulation, and advanced query analysis to extract meaningful insights.

> Tools & Technologies

- SQL (MySQL / SQL Server)
- CSV Dataset
- GitHub

> **Project Structure**

- "01_creating_database.sql" → Database creation
- "02_creating_table1.sql" → Table 1 creation
- "03_creating_table2.sql" → Table 2 creation
- "04_basic_queries.sql" → Basic SQL queries
- "05_join_queries.sql" → Joins (INNER, LEFT, etc.)
- "06_aggregation_queries.sql" → Aggregations (SUM, AVG, COUNT)
- "07_advanced_queries.sql" → Advanced SQL concepts
- "swiggy.csv" → Dataset

---

> ** Key Analysis Performed**

- Data cleaning and structuring
- Restaurant performance analysis
- Price and rating insights
- Joins between multiple tables
- Aggregation and grouping
- Advanced SQL queries

## Key Insights
- Identified top-performing restaurants based on revenue
- Analyzed peak order times and demand trends
- Evaluated customer ordering behavior
- Derived insights on delivery performance and efficiency
